# 消息格式转换

## 概述

QQ 消息通过 OneBot 协议以段（Segment）数组的形式到达，每个段有类型（type）和数据（data）。
网关需要将这些段转换为纯文本表示，再包装成 IRC PRIVMSG 发送。

## 段类型转换规则

| QQ 段类型 | IRC 表示 | 说明 |
|-----------|---------|------|
| text | 原文 | 直接透传 |
| face | [face:表情名] | 用方括号包裹表情描述文字 |
| image | ASCII art | 用 rascii_art 渲染为彩色 ASCII 画 |
| at | [at:昵称] | 解析 QQ 号为群名片或昵称 |
| 其他 | [类型名] | 未知类型用方括号包裹类型名 |

## 图片处理流程

1. 从段数据中提取 file 字段
2. 调用 OneBot API `get_image` 获取本地文件路径
3. 用 image 库打开并解码图片
4. 用 rascii_art 渲染为 15 行高的彩色 ASCII art
5. 用换行符包裹输出

**限制**：图片使用本地路径，网关必须和 OneBot 服务部署在同一台机器。

## 发送者名称解析

### 群消息发送者

1. 调用 `get_group_member_info` 获取群成员信息
2. 按优先级取名称：头衔+群名片 > 群名片 > 昵称 > QQ 号
3. 去除空白字符和标点符号

### 私聊发送者

1. 取发送者昵称
2. 去除空白字符和标点符号
3. 若结果为空，使用 QQ 号

## IRC 频道映射

| IRC 概念 | QQ 概念 | 格式 |
|----------|---------|------|
| 频道 #12345 | QQ 群 12345 | `#` + 群号 |
| 私聊目标 12345 | QQ 好友 12345 | QQ 号（无 `#` 前缀） |
